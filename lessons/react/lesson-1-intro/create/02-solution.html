Code Explanation

  • The index.html file is the entry point of the application. It contains our root div where our React app will be rendered and the script tag that loads the React library.
  • The main.js file is the entry point for our React code. It's only job is to render our App.js component into the root div in the index.html file.
  • The App.js file is the main component of our application. It is used to organize and structure our application's UI.
  • Inside of each Component:
    • We create a function that returns JSX code. The JSX code allows us to mix HTML and JavaScript.
    • We are able to call our components inside of other components. We can do so by <[Component-Name] />
    • Each component has its script tag loaded in the html file before our main.js file so that it has access to the component's logic.
index.html

  <!DOCTYPE html>
  <html lang="en">
  <head>
      <meta charset="UTF-8">
      <meta name="viewport" content="width=device-width, initial-scale=1.0">
      <title>React App</title>
      <script src="https://unpkg.com/react@18/umd/react.development.js"></script>
      <script src="https://unpkg.com/react-dom@18/umd/react-dom.development.js"></script>
      <script src="https://unpkg.com/@babel/standalone@7/babel.min.js"></script>
  </head>
  <body>
      <div id="root"></div>
      <script type="text/babel" src="/components/SelectedRecipes.js" defer></script>
      <script type="text/babel" src="/components/SavedRecipes.js" defer></script>
      <script type="text/babel" src="/components/RecipeForm.js" defer></script>
      <script type="text/babel" src="/components/Header.js" defer></script>
      <script type="text/babel" src="App.js" defer></script>
      <script type="text/babel" src="main.js" defer></script>
  </body>
  </html>
