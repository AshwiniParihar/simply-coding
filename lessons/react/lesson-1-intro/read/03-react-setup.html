Setting up a React App

There are several ways to set up a React app, but for our purposes, we will use Content Delivery Networks (CDNs) to import React and ReactDOM. We will also use Babel via a CDN. Babel is a JavaScript compiler that transforms JSX into standard JavaScript, making it understandable by the browser. While popular tools like Vite and Next.js are commonly used to create React applications, using CDNs allows us to get started quickly and easily without needing to install additional software or use the terminal.

Step 1: Creating the HTML File

First, create your HTML file (e.g., index.html) with the following basic structure: 


        <!DOCTYPE html>
        <html lang="en">
        <head>
            <meta charset="UTF-8">
            <meta name="viewport" content="width=device-width, initial-scale=1.0">
            <link rel="stylesheet" href="index.css">
            <title>Your React App</title>
        </head>
        <body>
            <div id="root"></div>
        </body>
        </html>

In this structure:

  • The <head> section includes a link to your CSS file and a title for your app.
  • The <body> contains a <div> with an ID of root, which will serve as the container for your React application.

Additionally, create a CSS file (e.g., index.css) to style your app. You can add your styles there to enhance the appearance of your application.

Step 2: Adding CDNs and Script for index.js

Next, we will add the CDNs for React, ReactDOM, and Babel, along with the script for index.js. Update the <head> section of your HTML file to include the following lines: 


        <head>
            <meta charset="UTF-8">
            <meta name="viewport" content="width=device-width, initial-scale=1.0">
            <link rel="stylesheet" href="index.css">
            <script src="https://unpkg.com/react@18/umd/react.development.js" crossorigin></script>
            <script src="https://unpkg.com/react-dom@18/umd/react-dom.development.js" crossorigin></script>
            <script src="https://unpkg.com/@babel/standalone/babel.min.js"></script>
            <script type="text/babel" src="index.js"></script>
            <title>Your React App</title>
        </head>

This setup imports React, ReactDOM, and Babel, allowing you to use JSX in your application. The line <script type="text/babel" src="index.js"></script> will link to your main JavaScript file where your React code will be defined.

Step 3: Understanding the index.js File

Now, create your index.js file. This will be the main entry point for your React application. Add the following code to your index.js file: 


        const root = ReactDOM.createRoot(document.getElementById("root"));
        root.render(<h1>Hello World</h1>);

Here’s what’s happening in this code:

  • Creating the root element: ReactDOM.createRoot(document.getElementById("root")) selects the <div> with the ID of root in your HTML. This is where your React app will be rendered.
  • Rendering the heading: root.render(<h1>Hello World</h1>) tells React to render the <h1> element inside the root element. This serves as a simple output for your React application.
Tip: You can replace the content inside the <h1> tags with any message you like to see it change in your application.
