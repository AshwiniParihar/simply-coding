Setting Up the HTML File

The `index.html` file is the entry point of our React app. This file includes links to React, ReactDOM, and Babel, allowing us to run React code directly in the browser.

Step 1: Create `index.html`

In the root directory, create a new file named index.html. Add the following code for the initial setup, including the root `div` element and links to React, ReactDOM, and Babel. 



<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>React App</title>
    <script src="https://unpkg.com/react@18/umd/react.development.js"></script>
    <script src="https://unpkg.com/react-dom@18/umd/react-dom.development.js"></script>
    <script src="https://unpkg.com/@babel/standalone@7/babel.min.js"></script>
</head>
<body>
    <div id="root"></div>
</body>
</html>

Creating and Adding main.js for React Rendering

The `main.js` file renders the `App` component into the root element. Weâ€™ll add this file to the HTML later once `App.js` is created.

Step 1: Create `main.js`

In the root directory, create a new file named main.js. Add the following code to render the `App` component. Leave the HTML file open to add this script once the `App` component is ready. 


// main.js
const root = ReactDOM.createRoot(document.getElementById("root"));
root.render(<App />);

Creating the App Component

The `App.js` file will serve as the main container for our app. It will include the components we will make next. Add the h1 in the `App` component to display the title of the shopping list. Add the `App.js` script to `index.html` after creating this file.

Step 1: Create `App.js`

In the root directory, create a new file named App.js. Define the `App` component to render each shopping list category. 


// App.js
function App() {
  return (
    <div className="App">
        <h1>Shopping List</h1>
    </div>
  );
}

Now, add the `App.js` and `main.js` scripts to `index.html`. 


<!-- index.html -->
...
<script type="text/babel" src="App.js" defer></script>
<script type="text/babel" src="main.js" defer></script>

Creating the FoodList Component

The `FoodList.js` file defines the "Food" category. After creating this component, add it to `index.html`.

Step 1: Create `FoodList.js`

In the `components` folder, create a new file named FoodList.js. Define a function component named `FoodList` that returns a `div` element with an `h2` element with the value Food and an unordered list. Add 4 items to the list, feel free to use the values in the example below or use your own. 


// FoodList.js
function FoodList(){
    return(
        <div>
            <h2>Food</h2>
            <ul>
                <li>Flour</li>
                <li>Eggs</li>
                <li>Sugar</li>
                <li>Milk</li>
            </ul>
        </div>
    );
}

Add `FoodList.js` to `index.html`. 


<!-- index.html -->
...
<script type="text/babel" src="/components/FoodList.js" defer></script>

Now lets add our FoodList component to the App component.


        // App.js
        function App() {
          return (
            <div className="App">
                <h1>Shopping List</h1>
                <FoodList />
            </div>
          );
        }
