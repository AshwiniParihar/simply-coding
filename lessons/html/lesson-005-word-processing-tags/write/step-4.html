Subscript and superscript are often used in scientific and mathematical contexts. Use the <sub> tag to write subscript text and the <sup> tag for superscript text.

  1. Add a <sub> tag to represent a subscript, such as H<sub>2</sub>O for water.
    Example Result: H2O
  2. Next, use a <sup> tag for superscript, like x<sup>2</sup> to show x squared.
    Example Result: x2 



    Water is represented as H<sub>2</sub>O. 
    The equation is x<sup>2</sup> + y<sup>2</sup>.
Pro Tip: Use <sub> and <sup> for formulas, equations, and footnotes to accurately present data.
