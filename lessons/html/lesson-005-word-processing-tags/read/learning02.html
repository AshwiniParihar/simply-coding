Examples and Usage:

  • <b>
    : Used to make text bold. Example: 
    <b>Important</b>
    renders as Important.
  • <strong>
    : Adds strong emphasis to text. Example: 
    <strong>Warning</strong>
    renders as Warning but also conveys importance to screen readers.
  • <i>
    : Italicizes text. Example: 
    <i>Italicized</i>
    renders as Italicized.
  • <em>
    : Emphasizes text with italics and semantic meaning. Example: 
    <em>Critical</em>
    renders as Critical.
  • <u>
    : Underlines text. Example: 
    <u>Underlined</u>
    renders as Underlined.
  • <s>
    : Strikes through text. Example: 
    <s>Cancelled</s>
    renders as Cancelled.
  • <sub>
    : Subscript text. Example: H
    <sub>2</sub>
    O renders as H2O.
  • <sup>
    : Superscript text. Example: x
    <sup>2</sup>
    renders as x2.
