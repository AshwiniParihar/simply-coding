To further enhance text formatting, you can add underlined and strikethrough styles. Use the <u> tag to underline text and the <s> tag to show strikethrough text, which typically indicates that content is no longer relevant.

  1. Insert an opening and closing <u> tag around a new line of text to underline it, such as <u>This text is underlined</u>.
  2. Below that, use an opening and closing <s> tag to strike through some text, like <s>This text has a strikethrough</s>.

Note: You can add these lines of code anywhere you want in your file, just remember they need to be inside your <body> tag. 


 
        <u>This text is underlined.</u>
        <s>This text has a strikethrough.</s>
Pro Tip: While <u> and <s> can add useful formatting, they are often less common in semantic HTML. Use them carefully, ensuring that the visual change accurately represents the intended meaning.
