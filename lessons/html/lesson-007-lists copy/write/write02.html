Step 2: Adjust to <strong> and <em> Tags

Now that you have added the <b> and <i> tags, let's adjust them to use semantic tags that convey more meaning. Replace the <b> tag with <strong> and the <i> tag with <em>. This change will ensure that your emphasis is recognized not only visually but also by assistive technologies like screen readers.

  1. In your code, find the line with the <b> tag and replace it with <strong>This is a strongly emphasized text.</strong>.
  2. Next, find the line with the <i> tag and replace it with <em>This text is semantically emphasized.</em>. 

<div">
    <p>
        <strong>This is a strongly emphasized text.</strong>
        <em>This text is semantically emphasized.</em>
    </p>
</div>
Reminder: Unlike <b> and <i>, which only change the text's appearance, <strong> and <em> provide additional meaning, making your HTML more accessible and meaningful to users and search engines.
