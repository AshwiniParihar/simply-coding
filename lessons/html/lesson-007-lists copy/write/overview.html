Overview: Creating Lists in HTML

In this section, you will create lists step-by-step using HTML. You'll start by building an unordered list for ingredients and then create an ordered list for instructions, following the guide on the left.

As you write the code, you'll see your progress in real-time, helping you understand how to use lists effectively to organize content on a webpage. By the end of this exercise, you'll be able to confidently create and format both ordered and unordered lists in HTML!
