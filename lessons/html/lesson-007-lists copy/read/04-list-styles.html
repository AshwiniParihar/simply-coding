Part 4: Adjusting List Styles

Lists in HTML are versatile, allowing customization to enhance visual appeal and improve user experience. You can modify list appearance by changing bullet styles in unordered lists or numbering styles in ordered lists.

Changing Bullet Styles for Unordered Lists

By default, unordered lists display bullet points. You can modify these bullet styles to circles, squares, or remove them entirely using the list-style-type property applied through inline CSS. 

<ul style="list-style-type: circle;">
    <li>Circle Bullet</li>
    <li>Circle Bullet</li>
</ul>

<ul style="list-style-type: square;">
    <li>Square Bullet</li>
    <li>Square Bullet</li>
</ul>

<ul style="list-style-type: none;">
    <li>No Bullet</li>
    <li>No Bullet</li>
</ul>
  • Circle Bullet
  • Circle Bullet
  • Square Bullet
  • Square Bullet
  • No Bullet
  • No Bullet

Adjusting Numbering for Ordered Lists

Ordered lists can be modified to use letters, Roman numerals, or custom start numbers using the list-style-type property. 

<ol style="list-style-type: lower-alpha;">
    <li>Step a</li>
    <li>Step b</li>
</ol>

<ol style="list-style-type: upper-roman;">
    <li>Step I</li>
    <li>Step II</li>
</ol>

<ol start="3">
    <li>Step 3</li>
    <li>Step 4</li>
</ol>
  1. Step a
  2. Step b
  1. Step I
  2. Step II
  1. Step 3
  2. Step 4
