Part 5: Embedding Lists Within Lists (Nesting)

You can create nested lists by placing one list inside another list item. This is useful for creating subcategories within a list.

Example: Nested Unordered Lists

Hereâ€™s how you can create a nested list within an unordered list: 

<ul>
    <li>Fruits
        <ul>
            <li>Apples</li>
            <li>Bananas</li>
        </ul>
    </li>
    <li>Vegetables
        <ul>
            <li>Carrots</li>
            <li>Peas</li>
        </ul>
    </li>
</ul>
  • Fruits
    • Apples
    • Bananas
  • Vegetables
    • Carrots
    • Peas

This example shows a list of categories with nested items beneath each category.

Example: Nested Ordered Lists

You can also create nested ordered lists, useful for outlining substeps within a main step: 

<ol>
    <li>Prepare Ingredients
        <ol>
            <li>Measure flour</li>
            <li>Crack eggs</li>
        </ol>
    </li>
    <li>Mix Ingredients
        <ol>
            <li>Combine dry ingredients</li>
            <li>Add wet ingredients</li>
        </ol>
    </li>
</ol>
  1. Prepare Ingredients
    1. Measure flour
    2. Crack eggs
  2. Mix Ingredients
    1. Combine dry ingredients
    2. Add wet ingredients

In this case, the main list contains steps for a recipe, while the nested lists contain substeps.

Part 6: Using CSS to Style Nested Lists

You can further enhance the appearance of nested lists using CSS. For example, you can add padding, margins, or different list styles for nested lists. 

/* Style for the main list */
ul {
    list-style-type: square;
    padding-left: 20px;
}

/* Style for the nested list */
ul ul {
    list-style-type: circle;
    padding-left: 20px;
}

This CSS example changes the bullet style of the main list to squares and the nested list to circles, creating a clear visual distinction.
