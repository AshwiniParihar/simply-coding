Part 2: Understanding Unordered Lists (<ul>)

An unordered list is used when the order of items is not important. It is created using the <ul> tag, with each item wrapped in <li> tags. The items appear with bullet points by default. 

<ul>
    <li>Apples</li>
    <li>Oranges</li>
    <li>Bananas</li>
</ul>

Use unordered lists for items where the sequence doesn’t matter, like a shopping list or a list of features on a product page.
