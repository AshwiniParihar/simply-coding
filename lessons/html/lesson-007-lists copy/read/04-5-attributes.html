Understanding Attributes in HTML

In HTML, attributes are used to provide additional information about elements. They modify the behavior, appearance, or functionality of elements and are always specified in the opening tag. Attributes consist of a name, an equal sign, and a value enclosed in quotation marks.

Structure of an Attribute

An attribute is composed of the following parts:

  • Name: The name of the attribute, which identifies what type of information it modifies.
  • Equal Sign (=): Separates the attribute name from its value.
  • Value: The specific information or property being set for the attribute.

Example of Multiple Attributes

The following example demonstrates how to use multiple attributes within a single HTML tag: 

<img
    src="image.jpg"
    alt="Sample Image"
    width="200"
    height="150"
>

In the example above, the img tag contains multiple attributes, each modifying a specific aspect of the image:

  • src: Specifies the URL of the image to be displayed.
  • alt: Provides alternative text for the image, improving accessibility.
  • width: Sets the width of the image in pixels.
  • height: Sets the height of the image in pixels.

Adding Multiple Attributes

You can add as many attributes as needed to an element, as long as they are separated by spaces. When using multiple attributes, remember:

  • Each attribute should have its own name and value.
  • Values should always be enclosed in double or single quotes.
  • Separate attributes with spaces within the same opening tag.

Understanding attributes is essential for customizing elements and making them more dynamic and accessible. Whether you are styling a list or adding an image, attributes provide the flexibility needed to control how elements appear and behave on your web pages.
