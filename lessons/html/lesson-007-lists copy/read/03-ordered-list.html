Part 3: Understanding Ordered Lists (<ol>)

An ordered list is used when the sequence of items is important. It is created using the <ol> tag, with each item wrapped in <li> tags. Ordered lists display items with numbers by default. 

<ol>
    <li>Preheat the oven to 350°F.</li>
    <li>Mix the ingredients.</li>
    <li>Bake for 25 minutes.</li>
</ol>

Use ordered lists for instructions or steps that must be followed in a specific order, like a recipe or a how-to guide.
