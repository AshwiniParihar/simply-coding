Part 7: Description Lists (<dl>)

In addition to unordered and ordered lists, HTML provides **description lists** for defining terms and their descriptions. These lists are useful for glossaries, FAQs, or any scenario where you need to pair terms with their definitions.

Structure of Description Lists

A description list is created using the <dl> tag. It contains <dt> tags for terms and <dd> tags for their descriptions. 

<dl>
    <dt>HTML</dt>
    <dd>HyperText Markup Language, used to structure web pages.</dd>
    <dt>CSS</dt>
    <dd>Cascading Style Sheets, used for styling web pages.</dd>
</dl>
HTML
HyperText Markup Language, used to structure web pages.
CSS
Cascading Style Sheets, used for styling web pages.

Use Cases for Description Lists

  • Glossaries: Define terms and concepts in a structured way.
  • FAQs: Pair questions with their answers clearly.
  • Definition Tables: Present items and their descriptions, such as product features or settings descriptions.

Example: FAQ Section

<dl>
    <dt>What is HTML?</dt>
    <dd>HTML stands for HyperText Markup Language, used to create web pages.</dd>
    <dt>What is CSS?</dt>
    <dd>CSS stands for Cascading Style Sheets, used for styling web pages.</dd>
</dl>
What is HTML?
HTML stands for HyperText Markup Language, used to create web pages.
What is CSS?
CSS stands for Cascading Style Sheets, used for styling web pages.

In this example, the description list is used for an FAQ format, where each term (question) is paired with a description (answer).
