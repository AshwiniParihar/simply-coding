Common Mistakes
  • Forgetting to use relative paths, which can cause broken links when moving files.
  • Using absolute paths in local files, which can make files inaccessible if the website URL changes.
  • Misusing ../ or ./ for parent and current directories, leading to incorrect paths.
Best Practices
  • Use relative paths for internal links to make files portable.
  • Use descriptive folder names and maintain a logical structure for ease of navigation.
  • Test paths after any file restructuring to ensure links are still valid.
