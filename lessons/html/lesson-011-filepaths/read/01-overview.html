Understanding File Paths in HTML

Welcome! In this lesson, you'll learn about file paths in HTML, which are essential for linking to other files like images, CSS, JavaScript, and documents. A proper understanding of file paths allows you to organize and access resources correctly within your projects.

Did You Know? File paths are like directions that tell the browser where to find files, enabling your website to load the correct resources.

Types of File Paths

In HTML, there are two primary types of file paths:

  • Absolute Path: An absolute path points to the exact location of a file on the internet or on your computer. On the internet they usually start with http:// or https://.
    Example: https://example.com/images/logo.png
  • Relative Path: A relative path points to a file's location relative to the current file's directory.
    Example: ../images/logo.png (moves up one level to find the "images" folder).
