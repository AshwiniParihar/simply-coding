• One <h1> Per Page:
    • Each page should have only one <h1> element, representing the main title.
    • This helps search engines and screen readers understand the main topic of the page.
  • Do Not Skip Heading Levels:
    • Maintain a logical flow by not skipping heading levels (e.g., do not jump from <h1> to <h3> without an <h2> in between).
    • This ensures a clear and accessible content structure.
