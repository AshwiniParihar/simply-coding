Instructions and Steps:

You will create a simple table of contents for a book using the heading elements to structure the hierarchy of the sections.

  1. Using the Editor Pane on the right, you will see the basic HTML structure for a new page. It should start with the <!DOCTYPE html> to declare the document type.
  2. Make sure and add a title in the <head><title> section (line 4) that represents the content of the page (what your page is about).
  3. Start the table of contents by using the <h1> tag for the main title of your book. Remember, the main content goes between the <body> tags.
  4. Structure the rest of the document using <h2> to <h6> tags, organizing it as you would a book's table of contents.
  5. Click the button in the editor to view your page.
