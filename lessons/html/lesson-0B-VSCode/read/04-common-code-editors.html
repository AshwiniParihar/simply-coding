• Visual Studio Code (VS Code): Known for its lightweight design and powerful features, it supports many languages and has an active ecosystem of extensions. This is what we will be using in class.
  • Sublime Text: A fast and highly responsive editor with powerful plugin support, known for its “Goto Anything” feature that helps navigate large codebases quickly.
  • Atom: Developed by GitHub, Atom is highly customizable and integrates directly with Git and GitHub for version control.
  • Notepad++: A simple and efficient code editor that supports various programming languages, often used for quick edits or small projects.
  • Brackets: An open-source editor specifically for web development, featuring live preview capabilities that allow you to see changes instantly in the browser.
  • Vim: A highly customizable, keyboard-focused editor that has a steep learning curve but is incredibly efficient for advanced users.
