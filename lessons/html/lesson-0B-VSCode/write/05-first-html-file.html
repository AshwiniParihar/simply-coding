1. In the VS Code Explorer panel on the left
    • Create a new folder named HTML inside of your Developer folder.
    • Create a new folder named CSS inside of your Developer Folder
    • Create a new folder named JavaScript inside of your Developer Folder
    • Create a new folder named Projects inside of your developer folder
    • You can use the "New Folder" icon (a folder with a "+" symbol) or right-click in the Explorer panel and select "New Folder"
  2. Then in your HTML folder create a new file. You can use the "New File" icon (a document with a "+" symbol) or right-click in the Explorer panel and select "New File".
  3. Name your file day-one.html and press Enter. This will create a new blank HTML file inside your folder.If you created your file in the wrong location(ensure it is in the HTML folder), you can click and drag it to the right location.
  4. Start writing your HTML code in the file. Below is an example to get you started: 
    

    <h1>Hello, World!</h1>
    <h2>This is my first HTML file.</h2>
    <p>My Name is Zachary.</p>
    <p>This is my first HTML file.</p>
  5. Save your file by pressing Ctrl + S (or Cmd + S on macOS).
