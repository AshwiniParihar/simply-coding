Basic Concepts

  • File Explorer: This is the program on your computer that lets you view, create, move, and delete files and folders.
  • Folder Structure: Folders are like containers that hold files. You can have folders inside folders to keep things organized.
  • File Types: Files can be text documents, images, code files, and more. Every file has a specific type based on its format, such as `.txt`, `.html`, or `.jpg`.
  • Creating Files: You can create a new file by right-clicking in a folder and choosing “New > Text Document” (or another file type).
  • Moving Files: Files can be dragged and dropped into different folders to organize your work.
  • Renaming Files: To rename a file, right-click it and choose “Rename,” then type the new name.
