Creating new files is a basic but essential task for coders. You can create files in several ways depending on the file type you need. For instance, to create a simple text file:

  • Navigate to the folder where you want to create the file.
  • Right-click in the folder and select “New > Text Document” (or the file type you need).
  • Name your file and press `Enter` to save it. Double-click the file to open it and start editing.
  • When naming your file there are two parts to include:
    • The name: myfile
    • The extension: .html
    • Full file name: myfile.html

Pro Tip: Always give your files clear and descriptive names. For example, instead of naming a file `test1.txt`, name it something like `ProjectDescription.txt` to make it easier to identify later.
