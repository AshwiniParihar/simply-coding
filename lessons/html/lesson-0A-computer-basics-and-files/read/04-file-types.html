Every file on your computer has a specific type, which tells the operating system and other programs what kind of data it holds. File types are indicated by their file extension (the part after the dot in the file name). Some common file types include:

  • .txt: Plain text file, great for notes or basic file creation.
    Example: `notes.txt`, `to-do-list.txt`.
  • .html: Hypertext Markup Language file, used to create webpages.
    Example: `index.html`, `about-me.html`.
  • .jpg/.png: Image files, used for photos and graphics.
    Example: `profile-picture.jpg`, `logo.png`.
  • .docx: Microsoft Word document.
    Example: `resume.docx`, `project-report.docx`.

You can change the file type when saving a file in most programs by selecting the format from a dropdown list. Be mindful of the file type you need when saving files, especially in coding, where using the correct file format (`.html`, `.css`, `.js`) is crucial for proper functionality.

Pro Tip: Enable file extensions in File Explorer to easily see the file type. To do this, click on the View tab at the top of File Explorer, then check the box for “File name extensions.”
