Renaming a file is important, especially as your projects evolve and you need to keep your files organized. To rename a file:

  • Right-click the file and select “Rename.”
  • Type the new name and press `Enter` to confirm.

Pro Tip: If you're renaming files that will be used in code, avoid spaces in the file names. Instead of `My File.txt`, use `MyFile.txt` or `my-file.txt`. Spaces can cause issues when linking to files in code.
