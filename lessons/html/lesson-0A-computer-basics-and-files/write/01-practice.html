Let’s Practice Managing Files

Follow these steps to practice the basics of navigating your computer and managing files:

  1. Open your File Explorer.
  2. Navigate to the "Student Data" folder on your computer.
  3. Find the specific folder for you: "Student Name".
  4. Navigate inside your student folder by clicking on it.
  5. Create a new folder in this folder called "Development"
  6. Create a new folder inside “Development” and name it `HTML`.
  7. Navigate inside the "HTML" folder and create a new file and call it `MyFirstFile.txt`.
  8. Open `MyFirstFile.txt`, type the sentence “This is my first file,” and save it.
  9. Navigate back to your "Development" folder.
  10. Create a new folder called "CSS".
  11. Locate your `MyFirstFile.txt file and move it to your "CSS" folder.
  12. Rename the file to `FirstRenamedFile.txt` by right-clicking and selecting “Rename.”
  13. Confirm that the file has been moved and renamed successfully.

Once you’ve completed all the steps, ensure that you can locate the file in its new location with the new name.
