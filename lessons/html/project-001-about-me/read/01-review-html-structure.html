  • HTML documents have a basic structure that starts with <!DOCTYPE html>, followed by the <html>, <head>, and <body> tags.
  • The <head> contains information about the page, while the <body> contains the visible content.
  • Notice: The HTML 'skeleton' often includes a title in the head and a paragraph in the body!

Remember: HTML code is read and rendered from the top to bottom. This means the web browser will read and build your code in the order it is written from the top of the page down.

Example Code:


<!DOCTYPE html>
<html lang="en">
    <head>
        <meta charset="UTF-8">
        <meta name="viewport" content="width=device-width, initial-scale=1.0">
        <title>My First HTML Page</title>
    </head>
    <body>
        <h1>Welcome to My Page!</h1>
        <p>This is a paragraph in the body of the HTML document.</p>
    </body>
</html>
