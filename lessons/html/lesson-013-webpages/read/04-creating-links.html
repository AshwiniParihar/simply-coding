Creating Links to Other Pages

In web development, links allow users to navigate between different pages of a website. You can link from one HTML file to another, like index.html (home) to about.html or contact.html, to build a multi-page site.

Linking to Pages Using Anchor Tags

Links are created using the <a> (anchor) tag. The href attribute specifies the file path or URL. To link to about.html from index.html, use a relative path like <a href="about.html">About</a>. 

      
<a href="about.html">About</a>

Creating a Simple Navigation Bar

A basic navigation bar helps users move between the pages on your website. You can create a simple nav bar using inline <a> tags for each page link.

Best Practices
  • Ensure all links are relative paths for internal pages.
  • Keep your nav bar consistent on all pages for better navigation.
