Project Folder Structure

Organizing your files correctly is essential for long-term success as a developer. In this lesson, you'll learn the fundamentals of structuring a web development project. Starting with a solid, organized structure not only makes your code more manageable but also sets the foundation for building a scalable and professional website.

Basic Structure


📂 my-website/ (Root Folder)
├── 📄 index.html (Home Page)
├── 📄 about.html (About Page)
├── 📄 contact.html (Contact Page)
├── 📂 css/
│   └── 📄 style.css
├── 📂 js/
│   └── 📄 script.js
└── 📂 assets/
    └── 🖼️ logo.png

This structure organizes your HTML files at the top level, with separate folders for CSS, JavaScript, and assets, each serving its own purpose.

Key Components Explained

index.html

The starting page, often the homepage of your website.

Additional HTML files like about.html and contact.html for different sections of your website.

...

A folder containing all of your css files to organize all of your styles used in your website.

...

...

Note: This is the very basic structure we use in Web Development. Next we will go in-depth about some of the different ways we can structure our code.

Best Practices
  • Use clear, descriptive names for all files to make them easy to identify.
  • Keep a consistent folder structure to make collaboration easier.
  • Organize CSS, images, and scripts in separate folders for better structure.
index.html

The main entry page, usually serving as the homepage of your site. You generally don't link to this page since you start here, but once on another page its important to be able to return to the Home Page.

Example of how to link to this file: 
<a href="index.html">Home</a>
Additional Pages

For any new page you have on your website you will have a new HTML file for that page. Some common pages we will create are About Me, about.html and Contact Us , contact.html.

These files will have seperate HTML code to set up their structure for their unique content.

Example of how to link to this file: 
<a href="about.html">About Us</a>
css/ Folder

The css/ folder contains all the CSS files, which define the styles used in your website. This structure keeps styling separate from content, making your code more modular and maintainable.

Example of how to link to this file: 
<link rel="stylesheet" href="css/style.css">
js/ Folder

The js/ folder holds all JavaScript files, which are used to add interactivity and logic to your website. This keeps behavior separate from structure and style.

Example of how to link to this file: 
<script src="js/script.js"></script>
assets/ Folder

The assets/ folder contains images, icons, and other media files. This keeps visual resources organized and separate from your HTML, CSS, and JavaScript.

Example of how to link to this file: 
<img src="assets/logo.png" alt="Logo">
