Lesson 13: Web Pages

This lesson covers:

  1. Introduction to Multi-Page Websites
  2. Project Folder Structure
  3. Creating Links to Web Pages
  4. Write: Build a Multi-Page Website
  5. Create: Build a Multi-Page Website

Here’s an example of a multi-page website structure:

          📂 my-website/
          ├── 📄 index.html
          ├── 📄 about.html
          ├── 📄 contact.html
          └── 📂 assets/
              ├── 📂 css/
              │   └── 📄 style.css
              └── 📂 images/
                  └── 🖼️ logo.png

Each HTML file represents a different page on your website, you will learn to link them together using anchor tags like this:

<a href="about.html">About Us</a>
