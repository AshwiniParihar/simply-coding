VS Code Shortcuts for HTML Boilerplate

Efficient coding in VS Code begins with knowing essential shortcuts. One of the most powerful shortcuts is using the `!` character to instantly create a full HTML boilerplate. Let’s dive into this and other shortcuts that streamline your workflow, including copy, paste, and save.

Generate HTML Boilerplate with `!`

Type ! and press Tab to generate a complete HTML boilerplate

This shortcut provides a structured starting point, including the DOCTYPE declaration, `html`, `head`, and `body` tags. It’s great for quickly setting up an HTML file with all necessary elements in place!

Copy, Cut, and Paste

These essential shortcuts allow you to quickly copy, cut, and paste text or code within your document, streamlining your workflow.

Copy

Copy the selected text or code to the clipboard for reuse elsewhere.

Shortcut : Ctrl + C
Shortcut : Cmd + C
  • Allows quick duplication of selected content
  • Helpful for moving code blocks or text snippets across the document
Cut

Remove the selected text or code and copy it to the clipboard for relocation.

Shortcut : Ctrl + X
Shortcut : Cmd + X
  • Moves content to clipboard while removing it from the document
  • Ideal for reorganizing sections of code or text
Paste

Inserts the content from the clipboard at the cursor’s current location within your document.

Shortcut : Ctrl + V
Shortcut : Cmd + V
  • Efficiently reinserts previously copied or cut content
  • Useful for duplicating or moving code within the document
Move Line Up/Down

Move the current line or selected lines up or down in the code.

Shortcut : Alt + Up/Down Arrow
Shortcut : Option + Up/Down Arrow
  • Quickly reorganize code for readability and flow
  • Works on one or multiple lines
Find and Find in All Files

Use these shortcuts to locate text within a file or search for it across all files in your project.

Find : Ctrl + F
Find : Cmd + F
Find in All Files : Ctrl + Shift + F
Find in All Files : Cmd + Shift + F
  • Find: Highlights occurrences of the specified text within the open file.
  • Find in All Files: Searches across all files in the project, showing results by filename and line number.
  • Helps quickly locate and edit instances of variables, functions, or text across the project.

Comment/Uncomment Line

Add or remove comments on the selected line or block of code. This is useful for testing code without deleting it.

Shortcut : Ctrl + /
Shortcut : Cmd + /

Save File

Save the current file in the editor to ensure all changes are updated on disk.

Shortcut : Ctrl + S
Shortcut : Cmd + S

Pro Tip: Keyboard shortcuts can be challenging to learn and memorize at first. In the long run becoming proficient with shortcuts saves you an incredible amount of time. Start at first by just using some of the very basic ones like Copy, Paste, Cut, and Save.
