What is HTML Boilerplate?

In this lesson, we will jump into the essential structure of an HTML document—known as the boilerplate. A strong foundation starts with understanding the basic elements every HTML page needs.

Did You Know? The term "boilerplate" originally referred to the steel plates used in steam boilers, which were prefabricated and consistently produced. In coding, "boilerplate" means a standard template or starting point that’s used repeatedly, ensuring every HTML document is readable and accessible for both browsers and users.

By now you have seen and used everything in the boilerplate, often refered to as the skeleton of any webpage we build. The HTML boilerplate is the essential tags that establish the structure of your webpage. Each tag has a specific purpose, from defining the document type to specifying the head and body sections.

The core tags in an HTML boilerplate include <!DOCTYPE html> to declare the document type, <html> to define the language, <head> for meta information, and <body> for your visible content.

Example of an HTML Boilerplate:

<!DOCTYPE html>
<html lang="en">
  <head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <meta name="description" content="A brief description of your web page">
    <meta name="keywords" content="HTML, CSS, JavaScript">
    <meta name="author" content="Your Name">
    <title>My Web Page</title>
  </head>
  <body>
    <!-- Your content goes here -->
  </body>
</html>

This structure serves as the starting point for your HTML projects.
