Introduction To HTML

This section will introduce you to what HTML is, covering the basics and structure of HTML. This is your first step into the web development world!

Here is a simple example of an HTML structure:

<!DOCTYPE html>
<html lang="en">
  <head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>My First Webpage</title>
  </head>
  <body>
    <h1>Welcome to My First HTML Page</h1>
    <p>This is a simple paragraph demonstrating the basics of HTML structure.</p>
  </body>
</html>

This basic structure includes the doctype, head, and body sections.
