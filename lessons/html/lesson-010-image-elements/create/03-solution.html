Code Explanation

  • The <h1> tag creates the main title: My Travel Gallery.
  • The <p> tag provides a brief description of the gallery’s content.
  • The <ul> tag defines an unordered list for the gallery items, with each <li> holding a <figure>.
  • Inside each <figure>:
    • The <img> tag includes src, alt, width, and height attributes to display the image with appropriate dimensions and descriptions.
    • The <figcaption> tag provides a caption below each image, adding context to the visual content.
