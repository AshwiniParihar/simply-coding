Review: The Image Tag

Let's review what you’ve learned about the <img> tag! This tag helps us embed images into our HTML documents, making web pages more visually appealing and informative.

1. Purpose of the <img> Tag

The <img> tag is a self-closing element used to add images to a webpage. It doesn't need a closing tag. Images can enhance the design and provide additional information to users. 


<img src="https://example.com/image.jpg" alt="A descriptive image" />

2. Essential Attributes

The two key attributes of the <img> tag are src and alt.

  • src: Specifies the path to the image file, either as a URL or a local file path.
  • alt: Provides alternative text for accessibility, displayed when the image can’t load. 

<img src="https://example.com/image.jpg" alt="Image description" />

3. Width and Height Attributes

Use the width and height attributes to control the image size in pixels, maintaining layout consistency and preventing layout shifts. 


<img src="https://example.com/image.jpg" alt="Image description" width="200" height="150" />

4. Optional Title Attribute

The title attribute provides additional context. It appears as a tooltip when a user hovers over the image. 


<img src="https://example.com/image.jpg" alt="Image description" width="200" height="150" title="Tooltip text here" />

5. Common Mistakes

Common Mistakes:
  • Forgetting the alt attribute, which is important for accessibility.
  • Not optimizing images for the web, leading to slower load times.
  • Using vague alt text, which may confuse users relying on screen readers.
  • Leaving out width and height attributes, causing layout shifts on page load.
  • Mixing up relative and absolute paths, which can break links if file locations change.

6. Complete Example Code

Here’s an example combining all the key attributes we’ve reviewed:
