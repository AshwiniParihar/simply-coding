What is the <figure> Tag?

The <figure> tag is a semantic HTML element used to group images, diagrams, or other media content with an optional caption. It’s especially useful when you want to associate an image with descriptive text, like in a gallery or article.

Basic Structure

<figure>
  <img src="https://sc-course-materials.s3.us-west-2.amazonaws.com/frontend-course/assets/images-global/paris.jpg" alt="Eiffel Tower at sunset" width="300" height="200" />
  <figcaption>Eiffel Tower at sunset in Paris</figcaption>
</figure>

The <figure> tag usually contains an <img> tag and an optional <figcaption> tag for captions.

Attributes and Nesting

<img> Tag

The <img> tag inside <figure> should include src and alt attributes. These define the image’s location and alternative text.

...
<figcaption> Tag

The <figcaption> tag, nested within <figure>, provides a caption that describes or adds context to the image. It’s optional but helps with accessibility and user experience.

Common Mistakes:
  • Forgetting the <figcaption> tag, which helps users understand the context of the image.
  • Misusing <figure> for purely decorative images; use it only when a caption adds value.
  • Adding multiple <figcaption> tags in a single <figure> element; each <figure> should only have one <figcaption>.
<img> Tag in <figure>

The <img> tag should include both src and alt attributes to specify the image source and alternative text.

Example: 
<img src="https://sc-course-materials.s3.us-west-2.amazonaws.com/frontend-course/assets/images-global/paris.jpg" alt="Eiffel Tower at sunset" />
<figcaption> Tag

The <figcaption> tag provides a caption for the <figure> content. It improves accessibility by describing the media’s context or purpose.

Example: 
<figcaption>Eiffel Tower at sunset in Paris</figcaption>
