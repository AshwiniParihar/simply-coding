Responsive Images

srcset and sizes attributes allow for responsive images that scale with the device. 

      <img src="https://sc-course-materials.s3.us-west-2.amazonaws.com/frontend-course/assets/images-global/retrotape.png" 
      srcset="https://sc-course-materials.s3.us-west-2.amazonaws.com/frontend-course/assets/images-global/retrotape.png 1x, https://sc-course-materials.s3.us-west-2.amazonaws.com/frontend-course/assets/images-global/retrotape.png 2x" 
      sizes="(max-width: 600px) 100vw, 200px" alt="Description" />

Best Practice: Browser selects appropriate image for the viewport.

The <picture> Element

Use the <picture> element to provide different sources based on screen size or features. 

      
<picture>
  <source srcset="https://sc-course-materials.s3.us-west-2.amazonaws.com/frontend-course/assets/images-global/retrotape.png" media="(min-width: 800px)" />
  <source srcset="https://sc-course-materials.s3.us-west-2.amazonaws.com/frontend-course/assets/images-global/retrotape.png" media="(max-width: 799px)" />
  <img src="https://sc-course-materials.s3.us-west-2.amazonaws.com/frontend-course/assets/images-global/retrotape.png" 
  alt="Description" />
</picture>

Best Practice: Provides fallback support and art direction.

usemap="#retromap" alt="Description" />
