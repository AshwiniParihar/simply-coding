Mastering the Image Element (<img>)

The <img> element is your key to adding visual flair to web pages. It employs src for the source and alt for accessibility.

<img>

What is the <img> Tag?

The <img> tag is your tool for embedding images into HTML, making your sites vibrant and informative.

  • Self-closing tag - no end tag needed
  • Supports various image formats (JPG, PNG, GIF, WebP)
  • Part of HTML since the beginning
{ }

Basic Structure

<img src="path/to/image.jpg" alt="Description" />
No closing tag required src and alt are required attributes

Common Image Formats

.jpg Photos
.png Graphics
.webp Optimized
.svg Vectors
src

Source Attribute

Directs to the image location.

src="images/photo.jpg" src="https://site.com/img.jpg"
alt

Alt Attribute

Ensures accessibility, describing the image in text.

alt="Red rose in bloom"
Best Practice: Be descriptive but concise
size

Width & Height

Defines the image's dimensions.

width="800" height="600" width="100%" height="auto"
Prevents layout shift during loading
i

Title & Loading

Additional attributes for enhanced functionality.

title="Hover text" Shows tooltip on hover
loading="lazy" Enables lazy loading
★

Best Practices

Optimization

Compress images and choose the right format

Responsive

Use srcset for responsive images

Accessibility

Always include meaningful alt text

Performance

Specify width/height to prevent layout shift
