What are Void Elements?

Void elements in HTML are elements that do not have closing tags or content inside them. They are self-closing, meaning they don't need a seperate closing tag.'

Basic Structure

Examples include <br>, <hr>, and <img>. They do not wrap content but are complete by themselves.

Common Mistakes:
  • Adding closing tags to void elements, which is not necessary and can cause validation errors.
  • Forgetting the self-closing slash in XHTML documents, although optional in HTML5.
  • Neglecting to use the alt attribute for <img> elements, which is critical for accessibility.
  • Using void elements where content is expected, leading to incorrect structuring of content.
<br> Element

The <br> element produces a single line break in text (carriage-return).

Example: 
<p>This is line one.<br />This is line two.</p>
<hr> Element

The <hr> element represents a thematic break between paragraph-level elements.

Example: 
<hr />
<img> Element

The <img> element embeds an image with attributes like src and alt.

Example: 
<img src="path/to/image.jpg" alt="A description of image" />
<input> Element

The <input> element is used to create controls for form input.

Example: 
<input type="text" name="username" />
<link> Element

The <link> element specifies relationships between the current document and resources.

Example: 
<link rel="stylesheet" href="styles.css" />
