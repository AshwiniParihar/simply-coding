Write: Create Your Anchor Element

Now, it's your turn to code! Follow the steps below to build an anchor element with various attributes. Be sure to complete each step as you go!

Step 1: Create a Basic Anchor Element

Go to line 10 in your HTML file and replace the placeholder X with a basic anchor element. This includes an opening <a> and a closing </a> tag. 

<a></a>
Tip: The anchor tag is the foundation of hyperlinks in HTML!

Step 2: Add an href Attribute

Add the href attribute to specify where the link should navigate. Set it to "https://example.com" for this example. 

<a href="https://example.com"></a>
Note: The href attribute defines the linkâ€™s destination.

Step 3: Add a target Attribute

Include the target="_blank" attribute to open the link in a new tab or window. 

<a href="https://example.com" target="_blank"></a>
Pro Tip: Opening links in a new tab keeps the user on your page while directing them to another resource.

Step 4: Add Link Text

Add visible text between the tags. Use "Visit Example" as the link text. 

<a href="https://example.com" target="_blank">Visit Example</a>
Did You Know? The text between the tags is the clickable part of the hyperlink!

Step 5: Add a Title Attribute (Optional)

Add a title attribute to display a tooltip on hover. For example, set title="Visit Example Website". 

<a href="https://example.com" target="_blank" title="Visit Example Website">Visit Example</a>
Accessibility Tip: Adding a title attribute improves the user experience by providing context to screen readers.

Step 6: Check Your Code

Combine all the elements to create a complete anchor tag. Your final code should look like this: 

<a href="https://example.com" target="_blank" title="Visit Example Website">Visit Example</a>
Final Check: Copy this code into your editor and see it in action!
