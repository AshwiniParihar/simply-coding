3. Global Attributes in Anchor Elements

In addition to specific attributes, anchor elements can have global attributes that are available to all HTML elements. These include:

  • id: Assigns a unique identifier to the element, allowing for targeted CSS styling or JavaScript interaction. Example: id="example-link"
  • class: Assigns one or more classes to the element for styling or JavaScript interaction. Example: class="link"
  • draggable: Allows the element to be draggable. Example: draggable="true"
  • style: Adds inline CSS to the element. Example: style="color: red;" 
<a href="https://www.example.com" id="example-link" class="link" draggable="true">Visit Example</a>

The example demonstrates how global attributes like id, class, and draggable can be used with anchor elements.
