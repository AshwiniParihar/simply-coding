2. Understanding Anchor Attributes

Anchor elements can have several attributes that modify their behavior. The most common attributes include:

  • href: Specifies the URL the link points to. Example: href="https://example.com"
  • target: Determines where the linked document opens.
    • _blank: Opens the link in a new tab or window.
    • _self: Opens the link in the same tab (default behavior).
    • _parent: Opens the link in the parent frame.
    • _top: Opens the link in the full body of the window.
  • rel: Provides relationship hints for the link. Example: rel="noopener noreferrer"
  • title: Adds a tooltip that appears when hovering over the link. Example: title="Visit Example" 
<a href="https://www.example.com" target="_blank" rel="noopener noreferrer" title="Visit Example">Visit Example</a>

This example uses multiple attributes to control link behavior and enhance accessibility and security.
