What is the <label> Element?

The <label> element in HTML is used to associate text with an input field. This is particularly important for accessibility, as it allows screen readers to identify the purpose of each input field for users who rely on assistive technology.

Basic Structure

      <label for="name">Name:</label> <input type="text" id="name" />
  • Rendered Example

The <label> element uses the for attribute, which should match the id of the associated input. This creates a connection between the label and the input, making it clear what data should be entered.

Pro Tip: Always use labels for form fields to improve accessibility, even if they are visually hidden.

Attributes Explained

for

The for attribute in the <label> tag should match the id of the associated input field, linking the two.

Accessibility

Labels make forms accessible to screen readers, allowing visually impaired users to understand form fields' purposes.

Try It Out!

        
<label for="userEmail">Email:</label> 
<input type="email" id="userEmail" placeholder="Enter your email" />
Don't Do This
  • Don’t skip the <label> element for inputs, even if using placeholders.
  • Don’t use non-descriptive labels, like “Input” or “Text.”
for Attribute

The for attribute on a <label> element associates it with an input by matching the id of the input. This helps screen readers connect the label to the input field.

Example: 
<label for="name">Name:</label> <input type="text" id="name" />
Accessibility Importance

Associating labels with inputs improves accessibility for visually impaired users, as it allows screen readers to announce what the input is for.

Example: 
<label for="email">Email Address:</label> <input type="email" id="email" />
