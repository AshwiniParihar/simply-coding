/* TODO Instructions for the Exercise */
/* 1. Targeting All Paragraphs (Element Selector) */
p {
color: red;
}

/* 2. Targeting Specific Class (Class Selector) */
.important {
font-size: 1.5rem;
}

/* 3. Targeting Unique ID (ID Selector) */
#main-header {
background-color: lightblue;
}

/* 4. Targeting Attribute Values (Attribute Selector) */
input[type="text"] {
padding: 10px;
}

/* 5. Using Universal Selector (Universal Selector) */
* {
text-align: center;
}

Explanation of Solutions

  • Element Selector: The p selector changes the color of all paragraphs to red.
  • Class Selector: The .important selector increases the font size of elements with the important class to 1.5rem.
  • ID Selector: The #main-header selector changes the background color of the header to lightblue.
  • Attribute Selector: The input[type="text"] selector adds 10px padding to all input elements of type text.
  • Universal Selector: The * selector centers all elements on the page.
