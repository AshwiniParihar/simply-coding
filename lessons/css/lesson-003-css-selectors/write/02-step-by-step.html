We'll walk through each selector with a detailed example. Below is the HTML file content along with an explanation of how the CSS selectors are used:

Step 1: Wrtie HTML code

Go to line 10 in your code where you see X. Replace X with this html code. 


        
            <h1 id="main-title">Welcome to My Webpage</h1>
            <p>This is a normal paragraph that will be styled with the element selector.</p>
            <p class="important">This is an important paragraph that uses the class selector.</p>
            <p>This is another paragraph to demonstrate the element selector.</p>
        
            <input type="text" placeholder="Type something here...">
            <input type="password" placeholder="Password field">
        
            <div class="important">This div also uses the important class and will have bold text.</div>

Step 2: Add Element selectors

On the code editor open css file and add p as a selector with property color and value as blue. 


        p { color: blue; }

Effect: All paragraph texts on the page will be displayed in blue.

Step 3: Add Class Selector.

Now next add .important as a class selector and give the property font-weight with bold. 


        .important { 
          font-weight: bold; 
        }

Effect: Elements with this class will have bold text.

Step 4: Add ID Selector

targets the element with the id="main-title" and increases its font size to 36px. 


        #main-title { 
          font-size: 36px; 
        }

Effect: Only the element with the specific ID will be affected, making it stand out with a larger font.

Step 5: Add Attribute Selector

targets all <input> elements that have type="text" and applies a 2px black border around them. 


        input[type="text"] { 
          border: 2px solid black;
         }

Effect: Text input fields will be distinguished by their border.

Step 6: Add Universal CSS

applies to all elements on the page, making their content center-aligned. 


        * { text-align: center; }

Effect: Every element on the page, including headings, paragraphs, inputs, etc., will have center-aligned text.

        <!DOCTYPE html>

            <meta charset="UTF-8">

        <body>
            <h1 id="main-title">Welcome to My Webpage</h1>

            <p class="important">This is an important paragraph that uses the class selector.</p>
            <p>This is another paragraph to demonstrate the element selector.</p>
        
            <input type="text" placeholder="Type something here...">
            <input type="password" placeholder="Password field">
        
            <div class="important">This div also uses the important class and will have bold text.</div>
        </body>
        </html>

CSS File:


        /* 1. Element Selector: Changes the color of all <p> elements to blue */
          p {
              color: blue;
          }
          
          /* 2. Class Selector: Makes text bold for elements with the class 'important' */
          .important {
              font-weight: bold;
          }
          
          /* 3. ID Selector: Increases the font size for the element with ID 'main-title' */
          #main-title {
              font-size: 36px;
          }
          
          /* 4. Attribute Selector: Adds a border around <input> elements with type="text" */
          input[type="text"] {
              border: 2px solid black;
          }
          
          /* 5. Universal Selector: Center-aligns all elements */
          * {
              text-align: center;
          }
