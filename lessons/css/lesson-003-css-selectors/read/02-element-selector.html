Element Selector

Definition
Targets all HTML elements of a specific type, such as <p>, <h1>, <div>, etc. Usage: Directly use the tag name to create an element selector.

Example:

p {
    color: red;
  }

Explanation:

In this example:
  • This CSS rule will change the text color of all <p> (paragraph) elements on the page to red.

HTML Example:

    
    <p>This is a paragraph with red text.</p>
    <p>Another paragraph that will also be red.</p>
