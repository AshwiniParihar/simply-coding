Attribute Selector

Definition
Targets elements based on the presence or value of an attribute. You can style elements that contain certain attributes or that match a specific attribute value.

Usage
  • Use square brackets [] to specify the attribute.

Example:


    input[type="text"] {
    border: 1px solid black;
  }

Explanation:

In this example:
  • This rule applies a black border to any <input> element with type="text".

HTML Example:

    
      <input type="text" placeholder="Text input with a border">
      <input type="password" placeholder="This won't be affected">
