ID Selector

Definition
Targets an element with a specific id attribute. An id should be unique and used only once per page to identify a single element.

Usage
  • Use a pound sign (#) followed by the ID name.

Example:

#header {
    font-size: 30px;
  }

Explanation:

In this example:
  • This rule will only apply to the element with the id attribute set to header, making its font size 30px.

HTML Example:

    
      <div id="header">This is the header section.</div>
<p id="header">This paragraph **won't** be affected because `id` attributes should be unique.</p>
