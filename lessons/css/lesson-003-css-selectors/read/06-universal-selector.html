Universal Selector

Definition
Targets all elements on the page. Itâ€™s commonly used for resetting or normalizing styles across all elements.

Usage
  • Use the asterisk *.

Example:


    * {
      margin: 0;
      padding: 0;
    }

Explanation:

  • This CSS rule will remove all default margin and padding from every element on the page. It's often used at the start of a CSS file to ensure consistency across browsers.

HTML Example:

    
      <h1>This heading will have no margin or padding.</h1>
      <p>This paragraph will also have no margin or padding.</p>
      <div>This div will be marginless and paddingless as well.</div>
