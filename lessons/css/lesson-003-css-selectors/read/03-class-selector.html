Class Selector

Definition
Targets elements that have a specific class attribute. Classes can be reused across multiple elements, making them great for applying the same style to different sections of a page.

Usage
  • Use a dot (.) followed by the class name.

Example:

.highlight {
    background-color: yellow;
  }

Explanation:

In this example:
  • All elements with the class highlight will have a yellow background.

HTML Example:

    
      <p class="highlight">This paragraph will be highlighted.</p>
      <div class="highlight">This div will also be highlighted.</div>
