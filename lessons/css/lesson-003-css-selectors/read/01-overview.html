Introduction to CSS Selectors

Welcome to Lesson 3! Today, we’ll dive into CSS Selectors, an essential skill for customizing the look and feel of web pages. Selectors allow us to apply styles to specific HTML elements so we can design our pages just the way we want. Let’s explore how these selectors work.

What is a CSS Selector?
CSS selectors are patterns that we use to select HTML elements on our webpage. They help us decide which parts of our HTML should be styled in specific ways.

How do CSS Rules Work?
Each CSS rule consists of a selector (which element to style) and properties (what styles to apply).

Example:

h1 {  
    color: blue;  
    font-size: 24px;  
    }

Explanation:

In this example:
  • h1 is the selector, meaning it will target all <h1> elements.
  • Inside the curly braces {}, we define the styles: color: blue; and font-size: 24px;

We can divide CSS selectors into five categories:

  • Element Selector
  • Class Selector
  • ID Selector
  • Attribute Selector
  • Universal Selector
