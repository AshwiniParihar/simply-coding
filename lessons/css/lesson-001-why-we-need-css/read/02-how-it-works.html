How CSS Works?

CSS (Cascading Style Sheets) works by applying styles to HTML elements, enhancing their visual presentation. It uses a combination of selectors, properties, and values to style specific elements on a web page.

Syntax

CSS syntax consists of selectors and declarations:

      p {
        color: blue;
        font-size: 16px;
      }

Selector (p): This targets all (paragraph) elements on the page.
Declaration Block: Contains two declarations:

              color: blue; changes the text color of the paragraphs to blue.
              font-size: 16px; sets the font size of the paragraphs to 16 pixels.
