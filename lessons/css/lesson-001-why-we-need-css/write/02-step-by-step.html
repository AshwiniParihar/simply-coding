Build Your First Project with CSS

Now it's time to practice what you’ve learned! Follow the step-by-step instructions to create an html structure with various tags, including headings, paragraphs, buttons with their CSS styling.

Step 1: Add a Title

Go to line 10 in your code where you see X. Replace X with an opening and closing <h1> heading with the title "Welcome to My Page". Within the h1 tag just put the styling for color and font-size. 


  <h1 style="color: blue; font-size: 24px;">Welcome to My Page</h1>

Step 2: Add Text Under the Title

On line 12, directly under the title, add a paragraph tag <p > with the description: “This is a paragraph with inline styling.” and within a p tag apply css with font-size and font-family. 


<p style="font-size: 16px; font-family: Arial;">This is a paragraph with inline styling.</p>

Step 3: Add a button with a text click me

On line 13, below the paragraph, add one button <button> with the text "click me" and within the button tag put css styling with background-color,color,padding. 


<button style="background-color: red; color: white; padding: 10px;">click me</button>
Final Check: Paste this code into your editor to see the full itinerary displayed on the page.

Problem with Inline CSS

  • Redundancy: Each element need its own style attrbute.If you wanted to change the font size for all the paragraphs,you would need to go each paragraph and update it manually.
  • Inconsistency: If you forget to apply a style to some elements, the design becomes inconsistent.
  • Hard to maintain: If the website grows larger, managing styles directly in HTML becomes complicated and inefficient.

To avoid these issues, we move the styling to a separate CSS file. This makes the HTML file much cleaner and easier to maintain.

html file


        <!DOCTYPE html> 
        <html lang="en"> 
        <head> 
            <meta charset="UTF-8"> 
            <meta name="viewport" content="width=device-width, initial-scale=1.0"> 
            <title>My Webpage</title> 
            <link rel="stylesheet" href="style.css"> 
        </head> 
        <body> 
            <h1>Welcome to My Page</h1> 
            <p>This is a paragraph with external styling.</p> 
            <button>Click Me</button> 
        </body> 
        </html>

css file(style.css)


        h1 {
            color: blue;
            font-size: 24px;
        }
        
        p {
            font-family: Arial;
            font-size: 16px;
        }
        
        button {
            background-color: red;
            color: white;
            padding: 10px;
        }
  • Point out that the HTML file now contains only the structure (content), while the CSS file contains the presentation (styling).
  • Emphasize how much easier it is to change the look of the webpage now. For example, to change the color of all the buttons, you only need to edit the button rule in the CSS file, instead of every button in the HTML.
