Purpose of adding CSS

CSS (Cascading Style Sheets) is used to style and layout web pages, separating content (HTML) from design. This helps enhance the appearance of web pages, making them more visually appealing and responsive.

Example of CSS Usage


        body {
          background-color: #f0f0f0;
          font-family: Arial, sans-serif;
      }

2. Selectors and Properties

CSS uses selectors to target HTML elements and applies properties to define styles.

  • Selectors:Define the element(s) to be styled (e.g., h1, .class, #id).
  • Properties:Define the actual style to apply (e.g., color, margin).

Example


        h1 {
          color: blue;
          font-size: 24px;
      }

3. Common Styling Attributes

Here are some frequently used properties:

  • color: Text color (color: red;).
  • background-color: Background color (background-color: yellow;).
  • margin/padding:Space outside/inside elements.
  • border:Defines border style, width, and color.

Example


        button {
          background-color: #3498db;
          color: #fff;
          padding: 10px 20px;
          border: none;
          border-radius: 5px;
          cursor: pointer;
      }

5. Common Mistakes

Common Mistakes:
  • One of the most common mistakes developers make is forgetting to include the link to the external CSS file in the <head> section of the HTML document. Without this link, the CSS styles will not be applied to the page, resulting in an unstyled or poorly styled webpage.

Suppose you have one css file with name style.css


        h1 {
          color: blue;
          font-size: 24px;
      }
      
      p {
          font-family: Arial;
          font-size: 16px;
      }
      
      button {
          background-color: red;
          color: white;
          padding: 10px;
      }

This is html File


        <!DOCTYPE html> 
         <html lang="en"> 
         <head> 
         <meta charset="UTF-8"> 
         <meta name="viewport" content="width=device-width, initial-scale=1.0"> 
         <title>My Webpage</title> 
         
!-- Missing link to the external CSS file --

         </head> 
         <body> 
         <h1>Welcome to My Page</h1> 
         <p>This is a paragraph with external styling.</p> 
        <button>Click Me</button> 
        </body> 
        </html>

Corrected Version with the CSS link


        <!DOCTYPE html> 
        <html lang="en"> 
        <head> 
        <meta charset="UTF-8"> 
        <meta name="viewport" content="width=device-width, initial-scale=1.0"> 
        <title>My Webpage</title> 
        <link rel="stylesheet" href="styles.css">
!-- Correctly linked external CSS file --

        </head> 
        <body> 
        <h1>Welcome to My Page</h1> 
        <p>This is a paragraph with external styling.</p> 
        <button>Click Me</button> 
        </body> 
        </html>

Explanation

  • The <link> tag with rel="stylesheet" and href="styles.css" ensures that the external CSS file is linked to the HTML document.
  • Without this link, any styles defined in the external CSS will not be applied, leading to an unstyled webpage.
Final Tip: Tip: Always double-check that the file path specified in href matches the actual location of your CSS file relative to your HTML document.
