Final Code

The following code demonstrates a complete, styled webpage with a clean and separated CSS approach:

html file


  

    <!DOCTYPE html>
    <html lang="en">
    <head>
        <meta charset="UTF-8">
        <meta name="viewport" content="width=device-width, initial-scale=1.0">
        <title>My Simple Webpage</title>
        <link rel="stylesheet" href="styles.css">
    </head>
    <body>
        <h1 class="heading">My Simple Webpage</h1>
        <p class="paragraph1">This is the first paragraph, styled using inline CSS.</p>
        <p class="paragraph2">This is the second paragraph with a different font and size.</p>
        <button class="button">Click Me</button>
    </body>
    </html>

CSS Code Explanation

  • The <h1> tag is styled to be centered (text-align: center), bold (font-weight: bold), and teal (color: teal).
  • The first <p> tag uses the Arial font family, sets the font size to 16px, and applies the styling to make it distinct from the second paragraph.
  • The second <p> tag uses a different font family, Georgia, and a larger font size of 18px, providing variation in style from the first paragraph.
  • The <button> tag is styled with a background color of #3498db (blue), white text color, padding, no border, and a cursor pointer.
    It has a smooth transition effect (transition: background-color 0.3s) on hover, where the background color darkens to #2980b9.

Complete CSS Code

css file


    /* Style for heading */
    .heading {
        text-align: center;
        font-weight: bold;
        color: teal;
    }
    
    /* Style for the first paragraph */
    .paragraph1 {
        font-family: 'Arial', sans-serif;
        font-size: 16px;
    }
    
    /* Style for the second paragraph */
    .paragraph2 {
        font-family: 'Georgia', serif;
        font-size: 18px;
    }
    
    /* Style for button */
    .button {
        background-color: #3498db;
        color: white;
        padding: 10px 20px;
        border: none;
        cursor: pointer;
        transition: background-color 0.3s;
    }
    
    /* Hover effect for button */
    .button:hover {
        background-color: #2980b9;
    }
