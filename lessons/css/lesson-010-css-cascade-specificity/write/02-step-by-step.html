Try to do this:

Step 1: Add div element

Go to line 10 in your code where you see X. Replace X <div> div element with the class container and apply external styles to it. 


        <div class="container">

Step 2: Add another div under the first div with ID

On line 12, directly under the first div, add a div tag <div> with an ID of main-container and apply some different styles. 


        <div id="main-container">

Step 3: Add a p paragraph tag

On line 13, Add a p paragraph tag with some inline css like color blue. 


        <p style="color: blue;">This is a paragraph.</p>

Step 4:Now in the css file add css code


        .container {
          width: 50%;
          background-color: lightblue;
      }

      #main-container {
          background-color: lightgreen;
      }

      p {
          color: black;
      }

      p {
          color: red !important; /* Will override the color black */
      }

Step 11: Full Code

Your full code should look like this:


        <!DOCTYPE html>
        <html lang="en">
        <head>
        <meta charset="UTF-8">
        <meta name="viewport" content="width=device-width, initial-scale=1.0">
        <title>CSS Cascade Example</title>
        </head>
        <body>
            <div class="container">
                <div id="main-container">
                    <p style="color: blue;">This is a paragraph.</p>
                </div>
            </div>
        </body>
        </html>

CSS code

        
        .container {
            width: 50%;
            background-color: lightblue;
        }

        #main-container {
            background-color: lightgreen;
        }

        p {
            color: black;
        }

        p {
            color: red !important; /* Will override the color black */
        }

Explanation

  1. The div with class .container will have a width of 50% and a light blue background.
  2. The div with ID #main-container will have a light green background.
  3. The paragraph inside will be red due to the !important in the CSS.
