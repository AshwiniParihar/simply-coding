HTML File


<!DOCTYPE html>
<html lang="en">
<head>
<meta charset="UTF-8">
<meta name="viewport" content="width=device-width, initial-scale=1.0">
<title>CSS Cascade Practice</title>
<style>
        /* Internal Styles */
        #box2 p {
            color: green;
        }
</style>
</head>
<body>
<div class="box1">
<p style="color: red;">This is the first paragraph.</p>
</div>
<div id="box2">
<p>This is the second paragraph.</p>
</div>
</body>
</html>

External CSS


  /* External Styles */
  .box1 {
      background-color: lightblue;
      width: 300px;
      height: 100px;
      margin: 20px;
  }

  #box2 {
      background-color: lightgreen;
      width: 300px;
      height: 100px;
      margin: 20px;
  }

Explanation of Solutions

  • box1 will have a light blue background, and the paragraph inside will be red due to the inline CSS.
  • box2 will have a light green background, and the paragraph inside will be green due to internal CSS.
