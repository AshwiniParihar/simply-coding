HTML File


    <!-- HTML -->
    <!DOCTYPE html>
    <html lang="en">
    <head>
        <meta charset="UTF-8">
        <meta name="viewport" content="width=device-width, initial-scale=1.0">
        <title>Font Styling Practice</title>
        <link rel="stylesheet" href="styles.css">
        <link rel="stylesheet" href="https://fonts.googleapis.com/css2?family=Roboto:wght@400;700&display=swap">
    </head>
    <body>
        <h1>CSS Font Properties Example</h1>
        <p>CSS makes it easy to change the size, weight, and style of fonts.</p>
    </body>
    </html>

CSS File


    body {
      font-family: 'Roboto', sans-serif;
      font-size: 16px;
  }
  
  h1 {
      font-size: 2.5rem;
      text-align: center;
      font-weight: 700;
  }
  
  p {
      font-size: 1rem;
      font-weight: 400;
  }

Explanation of Solutions

  • The page will display a large, bold heading that is centered.
  • The paragraph will appear in a regular weight, using the Roboto font with responsive size.
Key Takeaways:
  • You can try all the font property according to your choice.
