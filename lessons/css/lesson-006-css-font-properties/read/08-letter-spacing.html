Letter-spacing

The `letter-spacing` property controls the amount of space between characters. It can enhance readability or create unique text effects.

Example:

      
    <h2>Understanding `letter-spacing`</h2>
    <p style="letter-spacing: normal;">
    This text has normal letter spacing.
    </p>
    <p style="letter-spacing: 2px;">
    This text has a letter spacing of 2px.
    </p>
    <p style="letter-spacing: 0.5em;">
    This text has a letter spacing of 0.5em.
    </p>

OUTPUT:

Understanding `letter-spacing`

This text has normal letter spacing.

This text has a letter spacing of 2px.

This text has a letter spacing of 0.5em.
