Text-align

The `text-align` property specifies the horizontal alignment of text within an element. Common values are `left`, `right`, `center`, and `justify`.

Example:

        
      <h2>Understanding `text-align`</h2>
      <p style="text-align: left;">
      This text is aligned to the left.
      </p>
      <p style="text-align: center;">
      This text is centered.
      </p>
      <p style="text-align: right;">
      This text is aligned to the right.
      </p>
      <p style="text-align: justify;">
      This text is justified and spans the full width of its container.
      </p>

OUTPUT:

Understanding `text-align`

This text is aligned to the left.

This text is centered.

This text is aligned to the right.

This text is justified and spans the full width of its container.
