Font-weight

The `font-weight` property determines the boldness of the text. It can be set to values such as `normal`, `bold`, `bolder`, or numerical values like `100`, `400`, `700`, etc.

Example:

      
    <h2>Understanding `font-weight`</h2>
    <p style="font-weight: normal;">
    This text has a normal font weight.
    </p>
    <p style="font-weight: bold;">
    This text is bold.
    </p>
    <p style="font-weight: 700;">
    This text uses a font weight of 700.
    </p>

OUTPUT:

Understanding `font-weight`

This text has a normal font weight.

This text is bold.

This text uses a font weight of 700.
