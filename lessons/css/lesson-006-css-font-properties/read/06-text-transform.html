Text-transform

The `text-transform` property controls the capitalization of text. Common values are `uppercase`, `lowercase`, and `capitalize`.

Example:

      
    <h2>Understanding `text-transform`</h2>
    <p style="text-transform: uppercase;">
    This text is transformed to uppercase.
    </p>
    <p style="text-transform: lowercase;">
    THIS TEXT IS TRANSFORMED TO LOWERCASE.
    </p>
    <p style="text-transform: capitalize;">
    this text is transformed to capitalize each word.
    </p>

OUTPUT:

Understanding `text-transform`

This text is transformed to uppercase.

THIS TEXT IS TRANSFORMED TO LOWERCASE.

this text is transformed to capitalize each word.
