Adding Font Properties in HTML!

Welcome, students! Today, we will learn how to use CSS font properties to make text on webpages look attractive and readable.

Did You Know? You can control the appearance of text with CSS font properties such as font family, size, weight, style, and more.

CSS offers various properties that let you modify text to make it visually appealing and suitable for the design of your webpage. Here are some of the most important font properties:

  • font-family: Specifies the typeface of the text. Common choices include font-family: 'Arial', sans-serif; or font-family: 'Times New Roman', serif;. You can list multiple fonts to create a fallback in case one isn't available.
  • font-size: Sets the size of the text. You can use units like px, em, or rem. For example, font-size: 16px; or font-size: 1.5em;.
  • font-weight: Controls the boldness of the text. Use values such as font-weight: bold;, font-weight: 400; (normal), or font-weight: 700; (bold).
  • font-style: Changes the style of the text, such as font-style: italic; or font-style: normal;.
  • text-transform: Modifies the capitalization of the text. Options include text-transform: uppercase;, lowercase;, or capitalize;.
  • line-height: Specifies the spacing between lines of text. For example, line-height: 1.5; ensures better readability.
  • letter-spacing: Adds or reduces space between characters. Use letter-spacing: 1px; to create more space between letters.
  • text-alignThe text-align property specifies how the text is aligned within an element. Common values include left, right, center, and justify.

Here's an example of how to apply font properties to a paragraph: 

      
        p {
          font-family: 'Arial', sans-serif;
          font-size: 18px;
          font-weight: 400;
          font-style: italic;
          line-height: 1.6;
          text-transform: capitalize;
          text-align:center;
        }

By using these properties, you can make your text stand out, improve readability, and match the style of your overall webpage design.
