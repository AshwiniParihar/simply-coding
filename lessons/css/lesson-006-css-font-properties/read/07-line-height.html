Line-height

The `line-height` property sets the space between lines of text. It can be defined using numbers, percentages, or unit values.

Example:

      
    <h2>Understanding `line-height`</h2>
    <p style="line-height: 1.2;">
    This text has a line height of 1.2.
    </p>
    <p style="line-height: 1.5;">
    This text has a line height of 1.5.
    </p>
    <p style="line-height: 2;">
    This text has a line height of 2.
    </p>

OUTPUT:

Understanding `line-height`

This text has a line height of 1.2.

This text has a line height of 1.5.

This text has a line height of 2.
