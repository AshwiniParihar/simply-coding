Font-style

The `font-style` property specifies the style of the font. Common values are `normal`, `italic`, and `oblique`.

Example:

      
    <h2>Understanding `font-style`</h2>
    <p style="font-style: normal;">
    This text has a normal font style.
    </p>
    <p style="font-style: italic;">
    This text is italicized.
    </p>
    <p style="font-style: oblique;">
    This text has an oblique font style.
    </p>

OUTPUT:

Understanding `font-style`

This text has a normal font style.

This text is italicized.

This text has an oblique font style.
