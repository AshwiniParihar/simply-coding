Font-family

The font-family property specifies the typeface for text. It is a prioritized list of font names, where the browser tries to use the first available font in the list. Using multiple fonts ensures a fallback option if a particular font is not available on the user's system.

Example:

      
    <h2>Understanding `font-family`</h2>
    <p style="font-family: 'Times New Roman', serif;">
    This text uses the font family 'Times New Roman'. If it's not available, a generic serif font will be used.
    </p>
    <p style="font-family: 'Arial', sans-serif;">
    This text uses the font family 'Arial'. If it's not available, a generic sans-serif font will be used.
    </p>

OUTPUT:

Understanding `font-family`

This text uses the font family 'Times New Roman'. If it's not available, a generic serif font will be used.

This text uses the font family 'Arial'. If it's not available, a generic sans-serif font will be used.
