Review: Font Properties

Font properties in CSS are used to control the appearance of text on a webpage. Here’s a quick review of the key font properties:

1. font-family:

Specifies the typeface for the text. It's used to define a prioritized list of font names. If the first font isn’t available, the browser will try the next one. It’s good practice to specify a fallback font.

Example:


        font-family: 'Arial', sans-serif;

2. font-size:

Defines the size of the text. It can be specified in several units:

px(Pixels)

Fixed size.

em

Relative to the font size of the parent element.

rem

RRelative to the root element's font size

%

Relative to the parent element’s font size

3. font-weight

Specifies the thickness or boldness of the text. It can be:

  • Normal (normal), bold (bold), or a numeric value (100 to 900).

Example:


        font-weight: bold;

4. font-style:

Defines whether the text should be italicized or not.

  • Values: normal, italic, and oblique.

Example:


        font-style: italic;

text-align:

Controls the horizontal alignment of text inside an element.

  • Values: left, right, center, and justify.

Example:


        text-align: center;

line-height:

Sets the amount of space between lines of text, making text more readable.

Example:


        line-height: 1.5;

letter-spacing:

Adjusts the spacing between characters (also called tracking). Increasing it spreads out characters; decreasing it brings them closer together.

Example:


        letter-spacing: 1px;
Best Practices:
  • Use rem and em for consistent, scalable designs.
  • Always include a fallback font in the font-family property to ensure that if the custom font fails, the text will still render properly.
  • Avoid using px for font size unless fixed size is necessary.
  • Combine different font properties for a balanced and accessible design.
