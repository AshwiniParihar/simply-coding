What is the <img> tag?

The <img> tag is used to embed images into an HTML page. It is a self-closing tag, meaning it does not require a closing </img> tag.

Basic Structure

      <img src="path/to/image.jpg" alt="Description of the image" />

The <img> tag has two essential attributes: src (source) and alt (alternative text for accessibility).

Attributes Explained

src

The source URL of the image file.

<header>

Used for introductory content or navigational links.

<main>

Represents the primary content of the document.

<article>

Defines a self-contained piece of content that could stand alone.

<aside>

Holds content related to the main content, such as sidebars or ads.

Common Mistakes
  • Forgetting to use the alt attribute, which is essential for accessibility.
  • Using large, unoptimized images, which can slow down page loading.
  • Not specifying the image's dimensions (width and height), leading to layout shifts.
  • Providing non-descriptive alt text, which does not help screen readers.
Don't Do This
  • Don't skip the alt attribute. It's crucial for users relying on screen readers.
  • Don't link to local images from your computer; ensure images are hosted on a web-accessible server.
  • Don't hardcode image sizes for responsiveness. Use CSS or responsive attributes.
Best Practices
  • Always include the alt attribute to improve accessibility.
  • Optimize images to enhance web performance and loading times.
  • Test images across devices to ensure proper display and responsiveness.
  • Use descriptive filenames for image files to improve SEO.
Text Input

The text input allows users to enter a single line of plain text, such as names or general information.

Example: 
<input type="text" placeholder="Enter your name" />
Password Input

Password input masks characters to keep sensitive information secure while typing.

Example: 
<input type="password" placeholder="Enter your password" />
src Attribute

The src attribute specifies the image's file URL, allowing the browser to fetch and display the image.

Example: 
<img src="https://placekitten.com/200/300" alt="A small kitten" />
alt Attribute

The alt attribute provides text for screen readers and describes the image if it fails to load.

Example: 
<img src="https://placekitten.com/200/300" alt="A small kitten sitting on a pillow" />
width Attribute

The width attribute sets the image's width, either in pixels or as a percentage of the parent element.

Example: 
<img src="https://placekitten.com/200/300" alt="A small kitten" width="200" />
height Attribute

The height attribute sets the image's height, either in pixels or as a percentage of the parent element.

Example: 
<img src="https://placekitten.com/200/300" alt="A small kitten" height="300" />
title Attribute

The title attribute provides additional information about the image when hovered over, often as a tooltip.

Example: 
<img src="https://placekitten.com/200/300" alt="A small kitten" title="Cute kitten sitting on a pillow" />
