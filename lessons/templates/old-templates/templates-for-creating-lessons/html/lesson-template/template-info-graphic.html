What is the <img> tag?

The <img> tag is used to embed images into an HTML page. It is a self-closing tag, meaning it does not require a closing </img> tag.

Basic Structure

      <img src="path/to/image.jpg" alt="Description of the image" />

The <img> tag has two essential attributes: src (source) and alt (alternative text for accessibility).

Attributes Explained

src

The source URL of the image file.

...

Common Mistakes:
  • Forgetting to use the alt attribute, which is essential for accessibility.
  • Using images without ensuring they are optimized for web use, leading to slow loading times.
  • Not using descriptive alt text, which can affect user experience for those relying on screen readers.
  • Neglecting to set the dimensions (width and height) for images, which can lead to layout shifts during page loading.
  • Confusing relative paths with absolute paths. Relative paths link to files based on their location within the project directory, while absolute paths link to the full URL, which can break if the file location changes.
src Attribute

The src attribute specifies the image's file URL, allowing the browser to fetch and display the image.

Example: 
<img src="https://sc-course-materials.s3.us-west-2.amazonaws.com/frontend-course/assets/images-global/retrotape.png" alt="A small kitten" />
alt Attribute

The alt attribute provides text for screen readers and describes the image if it fails to load.

Example: 
<img src="https://sc-course-materials.s3.us-west-2.amazonaws.com/frontend-course/assets/images-global/retrotape.png" alt="A small kitten sitting on a pillow" />
width Attribute

The width attribute sets the image's width, either in pixels or as a percentage of the parent element.

Example: 
<img src="https://sc-course-materials.s3.us-west-2.amazonaws.com/frontend-course/assets/images-global/retrotape.png" alt="A small kitten" width="200" />
height Attribute

The height attribute sets the image's height, either in pixels or as a percentage of the parent element.

Example: 
<img src="https://sc-course-materials.s3.us-west-2.amazonaws.com/frontend-course/assets/images-global/retrotape.png" alt="A small kitten" height="300" />
title Attribute

The title attribute provides additional information about the image when hovered over, often as a tooltip.

Example: 
<img src="https://sc-course-materials.s3.us-west-2.amazonaws.com/frontend-course/assets/images-global/retrotape.png" alt="A small kitten" title="Cute kitten sitting on a pillow" />
