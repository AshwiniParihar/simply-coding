Welcome to Adding Images in HTML!

Welcome to the world of images in HTML! In this lesson, you'll learn how to enhance your webpages by adding images using the <img> element. Images make websites more engaging, informative, and visually appealing.

Did You Know? Images can be used to communicate complex ideas quickly and add visual context, improving user experience.
Pro Tip: Always use the alt attribute to improve accessibility and provide a description for the image if it fails to load.

Image Tag Infographic

Key Takeaways:
  • The <img> tag is essential for adding images to your web pages.
  • Always provide alt text for accessibility and SEO benefits.
  • Use the src attribute to link to your image files and ensure they are properly hosted.
  • Make use of width and height attributes to avoid layout shifts and improve page performance.
Best Practices:
  • Always include the alt attribute for accessibility and SEO.
  • Optimize images to reduce file size while maintaining quality for faster loading times.
  • Use descriptive filenames for images, as they can help with SEO.
  • Test your images to ensure they display correctly across different devices and screen sizes.
Don't Do This:
  • Don't use large images directly from your computer; always upload them to your server or use a reliable image host.
  • Don't hardcode the dimensions for images that should be responsive; instead, use CSS to manage their size appropriately.
  • Don't forget to check image links; broken links can lead to a poor user experience.
  • Don't use images solely for decoration without providing meaningful context in the alt attribute.
Common Mistakes:
  • Forgetting to use the alt attribute, which is essential for accessibility.
  • Using images without ensuring they are optimized for web use, leading to slow loading times.
  • Not using descriptive alt text, which can affect user experience for those relying on screen readers.
  • Neglecting to set the dimensions (width and height) for images, which can lead to layout shifts during page loading.
Additional Suggestions:
  • Consider using image formats like WebP for better compression and quality.
  • Use CSS to style images and maintain responsiveness across different layouts.

Code Example

Example Code: 
<img src="example.jpg" 
     alt="A descriptive text" 
     width="800" 
     height="600">

Quick Reference

Quick Reference:
Attribute Description
src Specifies the path to the image
alt Provides alternative text description
width Specifies the width of the image
height Specifies the height of the image

Troubleshooting

Troubleshooting:
Problem: Image not displaying
  • Check if the file path is correct
  • Verify the image file exists
  • Ensure proper file permissions
Problem: Image loading slowly
  • Optimize image size and format
  • Use appropriate dimensions
  • Consider using a CDN
