Review: Image Element Recap

Great work! Let's review what you've learned about the <img> element. This section will help solidify your understanding and ensure you're ready for the next step.

1. What is the <img> Tag?

The <img> tag is used to embed images in an HTML page. It is a self-closing element that requires attributes to work properly. 

<img src="https://placekitten.com/200/300" alt="A small kitten" />
Remember: The <img> tag does not have a closing tag.

2. Essential Attributes

The two essential attributes for the <img> tag are src and alt.

  • src: Specifies the URL of the image file.
  • alt: Provides alternative text when the image cannot be displayed.
<img src="https://placekitten.com/200/300" alt="A small kitten" />
Accessibility Tip: Always include the alt attribute for better accessibility.

3. Width and Height Attributes

You can control the size of the image using the width and height attributes, which define the dimensions in pixels. 

<img src="https://placekitten.com/200/300" alt="A small kitten" width="200" height="300" />
