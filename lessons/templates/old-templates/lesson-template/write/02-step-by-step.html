Build Your First Image Element

Now it's time to practice what you’ve learned! Follow the step-by-step instructions to create an image element that includes different attributes.

Step 1: Create a Basic Image Element

In the editor to the right, locate line 10. Replace the placeholder X with a basic image element. Use the <img> tag and add a placeholder image URL. 

<img src="https://placekitten.com/200/300" />
Tip: The <img> tag is self-closing, so you don’t need an ending tag!

Step 2: Add an Alt Attribute

Add an alt attribute to provide a text description of the image. This helps with accessibility and is displayed if the image fails to load. 

<img src="https://placekitten.com/200/300" alt="A cute kitten" />
Accessibility Tip: The alt attribute makes your image accessible to visually impaired users.

Step 3: Add a Width Attribute

Add a width attribute to set the image’s width in pixels. Set it to 200 pixels. 

<img src="https://placekitten.com/200/300" alt="A cute kitten" width="200" />
Pro Tip: Setting a fixed width helps manage the layout and loading behavior of your image.

Step 4: Add a Height Attribute

Add a height attribute to set the image’s height in pixels. Set it to 300 pixels. 

<img src="https://placekitten.com/200/300" alt="A cute kitten" width="200" height="300" />
Note: The height attribute helps ensure consistent aspect ratio and layout.

Step 5: Add a Title Attribute (Optional)

Add a title attribute to provide extra information about the image, displayed as a tooltip when the user hovers over it. 

<img src="https://placekitten.com/200/300" alt="A cute kitten" width="200" height="300" title="Cute kitten on a pillow" />
Did You Know? Adding a title can give users more context about the image.

Step 6: Check Your Code

Your complete image element should now look like this: 

<img src="https://placekitten.com/200/300" alt="A cute kitten" width="200" height="300" title="Cute kitten on a pillow" />
Final Check: Copy and paste this code into your editor to see it in action!
