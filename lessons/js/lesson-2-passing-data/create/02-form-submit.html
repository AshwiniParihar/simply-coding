Step 1: A Place for JS

Above the CLOSING body tag, create opening and closing script tags.

Note: The script tags could also go between the head tags, often you'll see them in the body section to improve the page load time and allow the user to see the page first.

Step 2: Event Listener

Between the script tags, use the id from your HTML button and add an event listener method to call a function.
Consider naming the new function 'saveMeal' or something descriptive.

Protip: the addEventListener allows for multiple events on a single element (mouseover, click, etc) whereas the onclick attribute only allows for a single event.

Step 3: Create your Function

Below your event listener, create the function to be called when your button is clicked. 

        document.querySelector("#ID").addEventListener('click', saveMeal);

        }

Step 5: Function TEST

Between the brackets of your new function, create an alert with a simple message, like 'I'm Working!'
Save your code, load it in the browser, click on your button and see if the alert appears!
If the alert shows up, your function is linked correctly to your button. Remove your alert code and move to the next step.

Remember: These simple tests along the way limit the amount of bugs to fix later!

Step 6: First Variable

Between the brackets of your function, create a variable targeting your text input HTML.
Also, store it's value and remove all white space using trim(). 

        function saveMeal(){
          let varName = document.querySelector("#ID").value.trim();
        }
Note: You can add multiple JavaScript command attributes to an element!

Step 7: Second Variable

Below your first variable, create a second variable targeting your textarea HTML.
Also, store it's value, remove white space using trim(), and use split passing in the comma delimmeter.

Remember: In JavaScript, the split() method is used to split a string into an array of substrings based on a specified separator! Like how you asked the user to comma separate their ingredients.

Step 8: If Statement

Below the variables, create an if else statement with condition that makes sure the user filled out each text box.

Hint: varName1 != '' && varName2 != '' - How is this code interpreted by JavaScript?

Step 9: Storage

Between the brackets of the if statement, add the localStorage command with the attribute that SAVES DATA.

Step 10: Passing Key and Data

Between the perenthsis of the localStorage setItem method, pass a KEY (which needs to be the value of the first text box entered) and pass the ingredients data.
Pass the ingredients data through JSON.stringify and add the JavaScript map command to trim each item from that string. 

        function saveMeal(){
          let varName = document.querySelector("#ID").value.trim();
          let varName2 = document.querySelector("#ID").value.trim().split(',');

          if(condition is here){
            localStorage.setItem(varName, JSON.stringify(varName2.map((item) => item.trim())));
          }
        }

Step 11: Clear the Form

Below the localStorage command, clear all text from the input box and textarea section by setting their values to an empty string.
You'll need to add two lines of code, one for each text box set to an empty string. 

        document.querySelector("#ID).value = '';
