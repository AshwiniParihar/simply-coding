Retrieving Data

Follow the steps to get your form working correctly to retreive the data from the local Storage and place it in the drop down menu!
The images to the right shows what this section helps you create.
Following the steps below!

Step 1: HTML Heading

First, we need to add some more HTML. In your HTML, create a heading tag with text asking the user which meal they'd like to select.

Step 2: Drop Down Menu

Below your HTML heading tag, add an HTML drop down menu with a single option as: Select a Meal.
Add an id attribute to the select tag set to a value of your choosing.

Step 3: New Function

In your JavaScript below the closing bracket of your function saveMeal (or whatever you named it), create a new function named something like loadMeals. 

      function saveMeal(){
        code in the function
      }
      
      function loadMeal(){
      
      }

Step 4: Call Your Function

Inside the if statement - below the alert code displaying the local storage, call your new function. 

      function saveMeal(){
        
        if(condition) {
          -- code --
          loadMeal();
        } else {
          -- code --
        }
      }
      
      function loadMeal(){
      
      }
Protip: We want to call our new function before the close of the if section so that we make sure the data gets into the local storage before we run this new function. We don't want it to run until we know everything worked correctly!

Step 5: Drop Down Variable

Between the brackets of your new function, create a variable set to your drop down element.

Step 6: Drop Down Variable

Using the variable name created above, add the innerHTML property to your variable and set it equal to "<option>Select a Meal</option>"

Note: The drop down will be populated with form submissions using the localStorage, so in order to always have the 'Select a Meal' text it's best to just hard code it in there.

Step 7: For Loop

Below the innerHTML variable property code, create a for loop that loops through every item in the local storage.

Remember: The length property will let us know how many times to loop through the local storage so we get all entries.

Step 8: Local Storage Key

Between the brackets of the for loop, create a variable equal to the local storage key function that looks at each index position. 

      function loadMeals(){
        variable code
        varName.innerHTML = ....

        for(let i = 0; i < localStorage.length; i++){
          const varName = localStorage.key(i);
        }
      }

Step 9: Retrieve the Data

Below the localStorage.key method code, retrieve the data using JSON.parse. Pass to it localStorage with its command to get data out and pass to that the variable name storing the local storage key name. 

      function loadMeals(){
        for(let i = 0; i < localStorage.length; i++){
          const varName = localStorage.key(i);
          JSON.parse(localStorage.[[CommandToRetrieveData]](varName));
        }
      }
Protip: You'll often see code instructions that need to be replaced within double square brackets...[[code]] just like above. Replace [[CommandToRetrieveData]] with the actual code used with the local storage to retrieve data.

Step 10: Create Element

Below the JSON.parse code, add a variable set to create an HTML option element.

Note: this code is inside the for loop so it will create an option tag for each item in the local storage.

Step 11: Option Value and Variables

Below the create element variable, use the create element variable name and add the 'value' property set to the localStorage key variable name.
Also, use the create element variable name and add the 'innerText' property set to the localStorage key variable name.
Also, use the drop down element variable name and add the appendChild property set to the localStorage key variable name. 

        const varName1 = document.getElementById("dropdown");
        varName1.innerHTML code.....
        
        for(let i = 0; i < localStorage.length; i++){
          const varName2 = localStorage.key(i);
          JSON.parse(...code here...);

          const varName3 = document.createElement("option");
          varName3.value = varName2;
          varName3.innerText = varName2;
          varName1.appendChild(varName3);
        }

Step 12: TESTING

Test it out! Save and refresh your page, fill out the form and see if it shows up in the drop down menu!

Step 13: Load the Drop Down

Above the closing </script> tag call the loadMeals() function!

Protip: when a visitor comes to your site for their first visit you to make sure that the drop down menu has everything from the local storage in there. By calling the loadMeals function outside of any other function event this will allow the loadMeals code to run when the page loads.
