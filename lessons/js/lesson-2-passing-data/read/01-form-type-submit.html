HTML Form Example and Explanation

The form below demonstrates the use of form tags and elements, including the submit button: 


      !! Notice the target="_blank" it will open the page /submit-form 
      and post data to it in a new page !!

      <form action="/submit-form" method="post" target="_blank">
        <label for="name">Name:</label>
        <input type="text" id="name" name="name" placeholder="Enter your name" />
        <br /><br />

        <label for="email">Email:</label>
        <input type="email" id="email" name="email" placeholder="Enter your email" />
        <br /><br />

        <button type="submit">Submit</button>
      </form>


<!-- If you click there it will open a new tab -->

Explanation of Form Structure

  • <form>: The container for form fields. It uses attributes like:
    • action: Specifies the URL where the form data will be sent. For example, action="/submit-form".
    • method: Defines the HTTP method for submitting data, typically post for modifying data or get for requests.
  • <input>: Defines various input fields, with types like:
    • text: Creates a text input field.
    • submit: A button that submits the form.
