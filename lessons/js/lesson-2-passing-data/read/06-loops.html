HTML Form Input: Text Field

An HTML form input allows users to enter data. A text field (<input type="text">) is used to collect single-line text input.

Example:

<form>
  <label for="name">Name:</label>
  <input type="text" id="name" name="name" placeholder="Enter your name">
  <button type="submit">Submit</button>
</form>

Key Attributes:

  • id/name: Identifies the input field.
  • placeholder: Displays hint text.
  • value: Sets a default input value.
  • maxlength/minlength: Limits input length.
  • required: Ensures input must be filled.

When the user clicks on the text field, it gains focus (indicated visually), enabling input.

Explanation of JavaScript Loops

JavaScript loops allow you to execute a block of code multiple times. Below are four commonly used loops:

1. for Loop

The for loop runs a block of code a specific number of times. 


for (let i = 0; i < 5; i++) {
  console.log(i);
}
// Output: 0 1 2 3 4
  • Use Case: When the number of iterations is known in advance.
  • Syntax: It contains three parts: initialization, condition, and increment.

2. forEach() Loop

The forEach() loop runs a function on each element of an array. 


const fruits = ['apple', 'banana', 'cherry'];
fruits.forEach(fruit => {
  console.log(fruit);
});
// Output: apple banana cherry
  • Use Case: When you need to perform actions on each array element.
  • Note: forEach() works only on arrays.

3. for...of Loop

The for...of loop iterates over iterable objects like arrays, strings, or sets. 


const letters = 'ABC';
for (const letter of letters) {
  console.log(letter);
}
// Output: A B C
  • Use Case: When you want to iterate over the values of an iterable object.
  • Note: It gives direct access to the elements, not their indexes.

4. for...in Loop

The for...in loop iterates over the keys (properties) of an object. 


const person = { name: 'Alice', age: 25 };
for (const key in person) {
  console.log(key + ': ' + person[key]);
}
// Output: name: Alice age: 25
  • Use Case: When you need to loop through an object's properties.
  • Note: Avoid using for...in on arrays as it loops through all enumerable properties, not just indexes.

Summary:

  • for: Use when you know how many times to loop.
  • forEach(): Use for arrays to execute a function on each element.
  • for...of: Use for iterating over values in iterable objects.
  • for...in: Use for looping over properties in an object.
