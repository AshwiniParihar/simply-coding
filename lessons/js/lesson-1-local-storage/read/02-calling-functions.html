JavaScript addEventListener() Explained

addEventListener() is used to attach an event handler to an element, making it respond to user interactions such as clicks, key presses, or mouse movements.

Syntax:


element.addEventListener('event', handlerFunction);

Click the button below to see addEventListener in action.

How it Works:

  • Select the element: Use getElementById to get the button element.
  • Add an event listener: Attach a 'click' event to the button.
  • Define a handler: When the button is clicked, the function updates the paragraph text.

Other Event Examples:

  • 'mouseover': Triggered when the mouse hovers over an element.
  • 'keydown': Triggered when a key is pressed on the keyboard.
  • 'submit': Triggered when a form is submitted.

Onclick Attribute

Using the onclick Attribute with an HTML Button

The onclick attribute in HTML is used to execute JavaScript code when a button (or other element) is clicked. This is a common way to make buttons interactive without requiring an external JavaScript file.

Example

In this example:

  • The onclick attribute value is set to call a function go().
  • When the button is clicked, the go() function runs, showing a pop-up message that says "Thank you!"

Key Points

  • Inline JavaScript: JavaScript code can be directly included within the onclick attribute.
  • Function Calls: You can also call external JavaScript functions (e.g., onclick="myFunction()").
  • Multiple Statements: Use semicolons to separate multiple statements (e.g., onclick="doSomething(); doSomethingElse();").

Differences?!

onclick attribute: Using onclick directly in an HTML element (e.g., <button onclick="myFunction()">) is a quick way to add a single, inline JavaScript function to run when the element is clicked. However, only one function can be assigned per element, and it mixes JavaScript with HTML.

addEventListener: Using addEventListener('click', myFunction) in JavaScript allows adding multiple functions to respond to a click event on the same element. It keeps JavaScript and HTML separate, enhancing code readability and flexibility.

Example:


    <!-- Using onclick attribute -->
    <button onclick="myFunction()">Click Me</button>

    <!-- Using addEventListener in JavaScript -->
    <button id="myButton">Click Me</button>
    <script>
      document.getElementById("myButton").addEventListener("click", myFunction);
    </script>

    
    <script>
      const button = document.getElementById("myButton");
      button.addEventListener("click", functionOne);
      button.addEventListener("click", functionTwo);
    </script>
