In this lesson, we will:

  • Create a page with two form fields for submitting meals and ingredients
  • The submitted meal will be saved to the browsers local storage
  • You'll also have a drop down to select each meal saved to the local storage
  • When selecting a meal in the drop down it's ingredients will be displayed
The JavaScript programming logic you'll use and learn will be:

  • Query Selector
  • Event Listeners
  • Functions, calling functions
  • Passing parameters, arguments
  • .value, .trim(), .split(), .length, .map(), .join(), .append
  • for loops, for each loops
  • localStorage, JSON.parse, JSON.stringify

By the end of the lesson you'll have a fully functional page to create and save meals with some persistence through the local storage!
