JavaScript Functions and Parameters Explained

A function in JavaScript is a block of code designed to perform a specific task. Functions can accept parameters, which are input values that customize how the function works.

Basic Syntax:


function functionName(parameter1, parameter2) {
  // Code to execute
  return parameter1 + parameter2;
}

Example with Parameters:

Enter two numbers and click the button to see the result.

How Functions Work with Parameters:

  • Parameters: Input variables defined in the function declaration (e.g., a and b).
  • Arguments: Actual values passed to the function when it's called (e.g., num1 and num2).
  • The function addNumbers takes two numbers as parameters, adds them, and returns the result.

Benefits of Functions:

  • Reusability: Write the code once and reuse it multiple times.
  • Modularity: Break down complex programs into smaller, manageable pieces.
  • Parameters: Make functions flexible by passing different inputs each time.
