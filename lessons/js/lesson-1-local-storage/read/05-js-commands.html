Explanation of JavaScript Commands

1. .value

Gets or sets the value of an <input> or <textarea> element. 


const inputValue = document.getElementById('inputField').value;
document.getElementById('inputField').value = 'New Value';

Use: Retrieve or modify user input in forms.

2. .trim()

Removes whitespace from the beginning and end of a string.


const text = '  Hello World!  '.trim();
console.log(text); // Output: 'Hello World!'

Use: Clean up user input by removing extra spaces.

3. .split()

Splits a string into an array based on a specified delimiter.


const words = 'apple,banana,cherry'.split(',');
console.log(words); // Output: ['apple', 'banana', 'cherry']

Use: Convert a string into an array of substrings.

4. .length

Returns the number of elements in an array or characters in a string. 


const arrayLength = ['a', 'b', 'c'].length;
console.log(arrayLength); // Output: 3

const stringLength = 'Hello'.length;
console.log(stringLength); // Output: 5

Use: Get the size of strings or arrays.

5. .map()

Creates a new array by applying a function to each element of an existing array. 


const numbers = [1, 2, 3];
const doubled = numbers.map(num => num * 2);
console.log(doubled); // Output: [2, 4, 6]

Use: Transform elements of an array without modifying the original.

6. .join()

Converts an array into a string, using a specified separator.


const fruits = ['apple', 'banana', 'cherry'];
const fruitString = fruits.join(', ');
console.log(fruitString); // Output: 'apple, banana, cherry'

Use: Combine array elements into a single string.

7. .append()

Adds content to the end of an element in the DOM.


const list = document.getElementById('myList');
list.append('New Item');

Use: Dynamically insert new content into a webpage.
